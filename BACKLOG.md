### [메인페이지]
- 로그인 후 사용자 개인대시보드가 아닌 메인페이지로 리다이렉트 [_0128/다윤/반영전_]
- 전체 서재데이터 출력되는 페이지 추가 [_0128/대석/반영전_]

### [개인 대시보드]
- 서재의 책은 페이지가 아닌 팝업이면 좋겠음 [_0128/다윤/반영전_]
- 캘린더 월/주 모두 볼 수 있게끔. 디폴트는 위클리. 현재는 월별로 되어있어서 모바일로 볼 경우 텍스트가 잘림. [_0128/다윤/반영전_]
- 캘린더 백그라운드 좌우 패딩 삭제 후 점캘린더로 변경  -> hover 동작에 tooltip 열기 [_0128/대석/반영전_]
- 서재 infinite scroll 문제 해결 [_0128/대석/반영전_]
- 월간 식탁 접속 횟수 / 시간 1~3위 뱃지 이팩트 (금, 은, 동) [_0128/대석/반영전_]

### [디깅박스]
- 디깅박스에 전달의 형식으로 url 전송할 경우 웹에 추가 안 되는 문제 해결 [_0128/대석/반영전_]
- 유튜브 등 링크 임베드 포함하도록 수정 [_0128/대석/반영전_]
- 디코에서 긁어 온 디깅박스 콘텐츠 내용에 링크 url의 헤더 타이틀 삽입 디코에 본문이 있을 경우 그 내용도 삽입 [_0128/대석/반영전_]
- infinite scroll 문제 해결 [_0128/대석/반영전_]

### [관리자페이지]
신청자 관리
   - 신청순서대로 나열&넘버링 -> 해당 정보 제거 [_0128/다윤/반영전_]
   - 서재활동 전체정보 -> 서재활동 이번달 서재활동(승인은 이번달 리뷰/프리뷰로 적용) [_0128/다윤/반영전_]
   - 추가 출력 데이터 -> 이벤트 신청시간, 코인사용여부 [_0128/다윤/반영전_]
   - 정렬방식 추가 -> 신청시간순, 순위순 [_0128/다윤/반영전_]
   - 순위 관리 알고리즘 생성 -> 예시 : 1. 정성적 활동(서재업로드 개수x3) 2. 활동량(식탁참여x2, 부깃아웃x1, 거실에 남긴 글 개수x1) 3. 선착순(7인안에 들면 가산점 +2) 과 같은 식으로 계산 [_0128/다윤/반영전_]
   - 식탁 접속 및 체류 관련 통계 페이지 [_0128/다윤/반영전_]
